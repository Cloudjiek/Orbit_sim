# Orbit

Simulation that calculates the orbital position (x, y), distance from the Sun, velocity, and gravitational force of a planet using Keplerian mechanics and Newton's method.

The code is divided into 5 parts:

1. Defining structs for a planet and the Sun
2. Calculating gravitational force using the Law of Universal Gravitation
3. Calculating velocity using the vis-viva equation
4. Calculating the eccentric anomaly E using Newton's method
5. Calculating the distance and orbital position (x, y) using the calculated values

IMPORTANT! The code has "raylib". If you're willing to visualize an orbit, you can uncomment it.
